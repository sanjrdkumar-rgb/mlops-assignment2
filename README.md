# MLOps Assignment 2 - IIT Jodhpur

## Project Description

This project demonstrates a text classification workflow using the DistilBERT transformer model from Hugging Face. The project includes model loading, tokenizer setup, Hugging Face Hub integration, Weights & Biases (W&B) logging, and repository management using GitHub. The goal is to build a reproducible MLOps pipeline for NLP model development and tracking.
